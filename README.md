# Church Charity Directory Browser Extension

This extension shows a one-click list of churches and charity numbers.
Click any church entry to open its exact Charity Commission page.

## Install on Linux (Chrome/Chromium/Brave)

1. Open your browser and go to `chrome://extensions`.
2. Turn on **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder:
   `/home/steve/Ai/CC/church-charity-directory/browser-extension`
5. Pin the extension to your toolbar.
6. Click the extension icon to view the list.
